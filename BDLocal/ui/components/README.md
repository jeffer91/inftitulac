# BDLocal/ui/components

Componentes reutilizables para BL.

Componentes previstos:

- tarjetas
- badges de estado
- tablas
- modales
- barras de progreso
- estados vacíos
- alertas

Regla: los componentes no deben conocer Firebase, Supabase, Excel ni Google Sheets. Solo reciben datos y devuelven HTML o actualizan DOM.
