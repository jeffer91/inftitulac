# BDLocal/data/mappers

Mapeadores de datos entre BDLocal y fuentes externas.

Función:

- convertir Excel a formato local
- convertir Firebase a formato local
- convertir Supabase a formato local
- convertir BDLocal a Firebase
- convertir BDLocal a Supabase
- convertir BDLocal a Excel
- convertir BDLocal a Google Sheets

Regla: aquí no se guarda ni se sincroniza. Solo se convierte el formato de los datos.
